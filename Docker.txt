## ğŸ“ Dockerfile (Simple Example)

```Dockerfile

# Set working directory
WORKDIR /app

# Copy files
COPY . .

# Install dependencies
RUN npm install

# Run the app

```

Build and run:

```bash

docker run -p 3000:3000 my-node-app
```

## ğŸ§  Bonus: Docker Compose (Optional)

If you have **multiple services** (e.g., app + database), use `docker-compose.yml`:

```yaml
version: '3'
services:
  web:
    image: nginx
    ports:
      - "8080:80"
  db:
    image: mysql
    environment:
      MYSQL_ROOT_PASSWORD: example
```

Run:

```bash
docker-compose up
```
